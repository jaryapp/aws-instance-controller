## ⚡ npm scripts

### 실행
```
git clone https://github.com/jaryapp/aws-instance-controller.git
cd aws-instance-controller
npm install
npm run dev

open webbrowser and access http://localhost:3000
```

### 빌드 & 배포
```
npm run build
npm start
```
